Command Aliases Usage Permission Description
/proton [argument] None /proton (help|update|reload) None Base command for Proton
/gamemode [argument] (player) /gm, /gmc, /gms, /gma, /gmsp /gamemode [creative|survival|adventure|spectator] (player) proton.gamemode Changes gamemode
/fly (player) None /fly (player) proton.fly Toggles flight mode
/tp [player] (player) None /tp [player] (player) proton.tp Teleports to a player (or a player to another)
/tphere [player] /s /tphere [player] proton.tphere Teleports a player to the sender
/tpall None /tpall proton.tpall Teleports all players to the sender
/tpcoords [integer] [integer] [integer] /tppos /tpcoords [x] [y] [z] proton.tpcoords Teleports the sender to the specified coordinates
/sethome None /sethome proton.sethome Sets the sender's home to the sender's location
/home None /home proton.home Teleports to the sender's set home location
/warp [string] (player) None /warp [warpName] (player) proton.warp, proton.warp.other & proton.warp.[warpName] Teleports the sender to a set warp's location
/setwarp [string] /warpset /setwarp [warpName] proton.setwarp Sets a warp location to the sender's location
/delwarp [string] /warpdel /delwarp [warpName] proton.delwarp Deletes an already set warp
/warps /warplist /warps proton.warps Views all currently set warps
